# Fusion Starter

Un portfolio moderne avec React, TypeScript et Express.

## Installation

1. Clonez le repository :
```bash
git clone <votre-repo-url>
cd fusion-starter
```

2. Installez les dépendances :
```bash
npm install
```

3. Lancez le serveur de développement :
```bash
npm run dev
```

## Scripts disponibles

- `npm run dev` : Lance le serveur de développement
- `npm run build` : Construit l'application pour la production
- `npm start` : Lance l'application en production
- `npm test` : Exécute les tests

## Configuration

Copiez `.env.example` vers `.env` et configurez vos variables d'environnement.

## Fonctionnalités

- Formulaire de contact avec Web3Forms
- Panel d'administration
- Upload d'images
- Newsletter avec double opt-in
- Portfolio avec études de cas
